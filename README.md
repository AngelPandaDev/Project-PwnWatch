# 🔐 PwnWatch — Breach Intelligence Dashboard

## 📌 Project Overview

**PwnWatch** is a cybersecurity-based web application that helps users check whether their email or credentials have been exposed in known data breaches. It provides insights into compromised data, evaluates risk levels, and suggests actions to improve digital security.

This project is being developed as part of a web development assignment to demonstrate **JavaScript concepts, API integration, and UI design skills**.

---

## 🎯 Purpose of the Project

The main goal of PwnWatch is to:

* Raise awareness about **data breaches and online security**
* Help users understand their **risk level**
* Provide **actionable recommendations** to secure their accounts

---

## 🌐 Public API Used

The application will use the:

* **Have I Been Pwned API**
  👉 Used to fetch real-time data about breached emails and exposed credentials

---

## 🚀 Planned Features

### 🔍 Search Functionality

* Users can enter an email address
* The system will fetch and display breach data

---

### 📊 Risk Scoring System

* Calculates risk level based on:

  * Number of breaches
  * Type of data exposed
* Displays result as:

  * Low / Medium / High Risk

---

### 🏷️ Filtering & Sorting

* Filter breaches based on:

  * Data types (passwords, emails, etc.)
* Sort results by:

  * Date
  * Severity

👉 Implemented using JavaScript **Array Higher-Order Functions**

---

### 🔐 Password Exposure Checker

* Checks if a password has been exposed in breaches
* Alerts users if it is unsafe

---

### 📈 Breach Analytics Dashboard

* Displays:

  * Total breaches
  * Data exposed
  * Trends and patterns

---

### 🌍 Organization-wise Breakdown

* Shows which companies/platforms were affected most

---

### 🧠 Security Recommendations

* Suggests actions like:

  * Change passwords
  * Enable 2FA
  * Monitor accounts

---

### 🛑 Action Panel

* “What should I do now?” section for immediate steps

---

### 🌙 Dark Mode UI

* Cybersecurity-themed interface with modern design

---

### 📡 Live Status Indicator

* Displays API status and last checked time

---

### 📁 Export Report

* Allows users to download breach reports

---

### 💾 Local Storage

* Save searched emails or preferences

---
